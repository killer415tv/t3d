Welcome to T3D Explorer

This is a Guild Wars 2 fansite

T3D Explorer lets you explore textured Guild Wars 2 map files in 3D, directly in your web browser. The purpose of this project is to allow a unique look into Tyria - the amazing world of Guild Wars 2. Some of the graphics are still missing such as animations, lighting, and various visual effects which give Guild Wars 2 its immersive experience.

Checkpoint Mode - Create & Edit

This version introduces a new Checkpoint Mode that lets you create, edit, and manage custom markers directly on the 3D map. Once a map is loaded, you'll see a toggle in the top-right corner to enable this mode.

How to use Checkpoint Mode:

  • Enable: Click the "Checkpoint mode" toggle in the top-right corner
  • Create: Move your mouse over the terrain - a green preview sphere shows where the marker will be placed. Press C to create a checkpoint
  • Default Radius: Use the "Default R" input field to set the default radius for new checkpoints
  • Select: Click on any existing checkpoint to select it (turns green). The previous checkpoint will return to blue.
  • Move: With a checkpoint selected, use keyboard controls:
    • A / D - Move along X axis (left/right)
    • W / S - Move along Z axis (forward/back)
    • Arrow Up / Down - Move along Y axis (up/down)
  • Edit Label: Press L to edit the checkpoint label (e.g., START, END, CP 1)
  • Edit Note: Press N to add a note to the checkpoint
  • Edit Radius: Press R to change the radius of the selected checkpoint
  • Resize (keyboard): Arrow Left decreases radius, Arrow Right increases it
  • Delete: Press X to delete the selected checkpoint
  • Export: Click "Download CSV" for CSV format or "Download XML" for XML checkpoint format

Tip: The bottom-left panel shows all your checkpoints with their XYZ coordinates in both 3D units and game units.

Beetlerank Checkpoint Viewer

You can also load checkpoint files from beetlerank.com to visualize them on the map. Use the CSV file input in the map selection screen.

Real-Time Telemetry

T3D Explorer is now connected to WebSocket to display the real-time position of users who are running telemetry programs. This allows you to see where other players are in the game world as they play.

Telemetry Programs

To use the real-time telemetry feature, you need to install one of these telemetry programs:

  • GW2Telemetry by Girbilcannon
  • gw2_telemetry by killer415
